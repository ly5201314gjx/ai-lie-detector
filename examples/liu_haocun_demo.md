# 🎬 实战示例：刘浩存 — 演技转型言论一致性分析

## 背景

基于之前对刘浩存的全面研究，本示例演示如何使用AI测谎仪技能检测她在**演技转型**话题上的言论一致性。

## 分析命令

```
📌 调用技能:
use_package('lie_detector_ai')

📌 快速扫描:
快速扫描刘浩存近2年关于演技转型/突破的言论一致性

📌 完整分析:
完整分析刘浩存2022-2026年期间关于"演技成长"和"角色类型"的言论一致性，输出full_report格式

📌 事件前后对比:
分析刘浩存在2021年父母事件前后的公开言论变化
```

## 预期分析维度

### 话题识别
系统会自动从收集的言论中聚类以下话题:
- **演技转型**: 从"谋女郎"标签到独立影后的演进
- **角色选择**: 被指"角色同质化"到"挑战复杂角色"的转变
- **争议回应**: 针对父母事件的不同表态
- **职业规划**: 对电影/电视剧发展的看法

### 矛盾检测示例

**潜在的检测点**:
1. 早期(2020-2021)强调"感谢张艺谋导演培养" vs 后期(2025-2026)强调"独立突破" — 分析是否构成立场转变
2. 对"只想演电影" vs "开始尝试电视剧"的表述一致性
3. 对网络争议从"不回应"到"正面面对"的变化轨迹
4. 对《想飞的女孩》获奖前后的评价变化

### 语言特征分析重点
- 早期采访中的"标准话术"密度
- 面对争议时的回避性语言使用
- 获奖后自信程度在语言上的体现

## 实际跑分建议

建议分别测试以下场景:
1. **单话题深度扫描**: 仅针对"演技转型"话题做深挖
2. **全量扫描**: 不限定话题，全自动检测
3. **事件驱动分析**: 以2021年父母事件为锚点做前后对比
4. **多人对比**: 刘浩存 vs 周冬雨 vs 章子怡（同为谋女郎的言论对比）
