## 场景一：快速扫描某公众人物近1年的言论

### 输入
```
快速扫描马斯科关于AI安全的言论一致性
```

### 系统执行流程
1. **意图解析**: 检测到关键词"快速扫描" → 选择 quick_scan 模式
2. **目标识别**: "马斯科" → 自动关联人物和话题
3. **数据采集**: 多引擎搜索"Elon Musk AI safety interview/statement"
4. **矛盾检测**: 检测对AI安全风险的立场变化
5. **报告生成**: 输出快速扫描报告

### 预期输出示例
```
🔍 AI测谎仪 · 快速扫描

目标: Elon Musk (马斯科)
评分: 🟡⚠️ 55/100
语言风险: 🔴 高风险
矛盾数: 4 处

核心矛盾:
1. AI安全风险立场 → 从"最大风险"到"相对乐观"的转变
2. AI监管态度 → 从"立即监管"到"警惕过度监管"
3. 对OpenAI评价 → 从联合创始人到批评者
4. 对AGI时间预测 → 从5年到10年再到"难以预测"
```

## 场景二：完整多话题深度分析

### 输入
```
全面分析余承东近3年关于"智能驾驶"的言论一致性，输出完整报告
```

### 系统执行流程
1. 多源采集: 发布会+采访+微博+年度演讲
2. 话题聚类: 
   - L3级自动驾驶时间表
   - 华为智驾 vs 特斯拉FSD
   - 技术路线选择（激光雷达 vs 纯视觉）
3. 6轮矛盾检测 + 语言分析
4. 输出完整报告

## 场景三：事件前后对比

### 输入
```
以刘浩存2021年父母事件为分界点，对比分析事件前后12个月的言论变化
```

### 系统执行流程
1. 设置锚点: 2021年7月（事件爆发时间）
2. 划分窗口: 事件前12个月 + 事件后12个月
3. 分别分析两个窗口的:
   - 言论数量/频率变化
   - 话题重心转移
   - 语言模式变化（从开放到回避）
   - 立场漂移检测
4. 输出对比报告

## 场景四：多人对比矩阵

### 输入
```
对比分析雷军、余承东、李想三人关于"造车"的言论一致性
```

### 系统执行流程
1. 多目标并行分析
2. 提取共同话题集
3. 生成对比矩阵
4. 标注每个话题上各人的立场稳定性

## 场景五：单话题深度追踪

### 输入
```
追踪分析华为任正非近5年关于"芯片"的所有公开言论
```

### 系统执行流程
1. 设置窄话题: "芯片"领域 + 子话题（麒麟芯片/供应链/替代方案）
2. 深度采集: 扩大搜索深度（更多页、更多关键词变体）
3. 细粒度时间线: 精确到月
4. 检测随外部事件（制裁升级）的立场变化
